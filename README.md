**菜单栏切换模式**

不习惯快捷键也没关系。Witch 支持菜单栏模式，点开顶部图标，所有窗口列在下拉菜单里，一眼全看到，一点就切换。

**多套切换方案**

可以同时配置多个切换器，分别对应"切换应用"、"切换窗口"、"切换标签页"等不同场景，每套方案独立设置快捷键、排序方式，完全按自己习惯来。

**个性化外观**

切换面板的大小、配色、字体、字号都可以调。既支持横向布局（模仿系统原生切换器风格），也支持竖向列表（更适合窗口多的场景）。

如果某个应用里的快捷键和 Witch 有冲突，可以把它加进排除列表，Witch 就不会在那个应用里激活，非常省心。

**Spring-Load 自动展开**
